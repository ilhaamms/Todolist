# Website Todolist

Ini adalah website Todolist, di web ini kita bisa melakukan seperti Menambah Todo, Complete Todo, dan menghapus Todo. Web ini juga sudah terintegrasi dengan database MYSQL

# Requirement

init `.env` file : `cp .env.example .env`
